If you see this page you get the Reveal.js tool with Bonita look and feel. To build a version of the training material:

  • Get a tagged version of the content (index.html + images) from Subversion (e.g. version 7.0.0.0 of BPAD)
  • Clone the training presentation tool Git repository
  • Copy and overwrite index.html and images/* in reveal.js folder
  • Go to reveal.js folder and run grunt zip
  • Rename reveal-js-presentation.zip to BPAD-7.0.0.0-presentation.zip
