If you see this page you get the Reveal.js tool with Bonita look and feel. To build a version of the training material:

  • Get a tagged version of the training content from Subversion (e.g. svn export https://service.svn.bonitasoft.org/training/materials/tags/BPAD-en-7.0-1.0.0)
  • Go to sources folder and run mvn -Dtranslation=us-EN clean install
  • Get the files from dist/target folder
